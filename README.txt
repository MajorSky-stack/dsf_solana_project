Solana Priority Fee Prediction

File Overview

SQL_queries.ipynb: Data extraction from Snowflake.

main.ipynb: Main modeling notebook.

data_5k.csv: Sample dataset (5,000 blocks).
Will be generated by SQL_queries.ipynb.

data_100k.csv: Full dataset (100,000 blocks) for complete analysis.
Generated by SQL_queries.ipynb with larger block range.

seasonality_uncomplete.csv: Sample hourly seasonality data for priority fees (5000 blocks).
Generated by SQL_queries.ipynb.

seasonality_complete.csv: Full dataset (1y+) Hourly seasonality data for priority fees.
Will be generated by SQL_queries.ipynb with larger block range.
